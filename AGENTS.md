# Agent Instructions

## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues in this repo. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles mapped to GitHub labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo with `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
